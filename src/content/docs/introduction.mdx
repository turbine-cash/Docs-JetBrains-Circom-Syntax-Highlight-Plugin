Full-featured Circom language support for IntelliJ IDEA and all JetBrains IDEs.

## Quick Start

Get up and running with Circom in your JetBrains IDE in under a minute:

1. **Install the Plugin**
   Open your JetBrains IDE → Settings → Plugins → Marketplace → Search "Circom Language Support" → Install

2. **Open a Circom File**
   Open any `.circom` file in your project

3. **Start Coding**
   Enjoy syntax highlighting, navigation, and all IDE features

## What is Circom?

[Circom](https://docs.circom.io/) is a domain-specific language for writing arithmetic circuits used in zero-knowledge proofs. It's widely used in blockchain projects for privacy-preserving computations and verifiable computation.

This plugin brings professional IDE support to Circom development, making it easier to write, navigate, and refactor your circuits.
