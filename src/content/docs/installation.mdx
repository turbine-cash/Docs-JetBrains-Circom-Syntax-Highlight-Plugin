## Installing from JetBrains Marketplace

The recommended way to install the plugin is through the JetBrains Marketplace.

<Steps>

1. **Open Plugin Settings**

   In your JetBrains IDE, go to **Settings** (or **Preferences** on macOS) → **Plugins**

2. **Search for the Plugin**

   Click on the **Marketplace** tab and search for "Circom Language Support"

3. **Install**

   Click the **Install** button next to the plugin

4. **Restart IDE**

   After installation, restart your IDE to activate the plugin

</Steps>

<Aside type="tip">
You can also install the plugin directly from the [JetBrains Marketplace website](https://plugins.jetbrains.com/) by clicking "Install to IDE".
</Aside>

## Verifying Installation

After installation, you can verify the plugin is working:

1. Open any `.circom` file
2. You should see syntax highlighting applied
3. Check **Settings** → **Editor** → **Color Scheme** → **Circom** to customize colors

## Requirements

- JetBrains IDE version **2024.3** or later
- Works with: IntelliJ IDEA, WebStorm, PyCharm, RustRover, CLion, GoLand, PhpStorm, and all other JetBrains IDEs

## Updating the Plugin

The plugin will automatically notify you when updates are available. You can also manually check for updates:

1. Go to **Settings** → **Plugins**
2. Click the **Updates** tab
3. If an update is available, click **Update**

## Troubleshooting

If the plugin doesn't appear to be working:

- Ensure your IDE version is 2024.3 or later
- Try **File** → **Invalidate Caches / Restart**
- Check that the plugin is enabled in **Settings** → **Plugins** → **Installed**

See the [Troubleshooting](/troubleshooting) page for more help.
