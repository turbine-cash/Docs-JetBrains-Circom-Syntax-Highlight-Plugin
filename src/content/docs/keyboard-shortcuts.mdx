A comprehensive reference of keyboard shortcuts for working with Circom files in JetBrains IDEs.

## Navigation

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Go to Symbol | `Cmd+O` | `Ctrl+Shift+N` |
| Go to Definition | `Cmd+B` or `Cmd+Click` | `Ctrl+B` or `Ctrl+Click` |
| Go to Line | `Cmd+G` | `Ctrl+G` |
| Navigate Back | `Cmd+[` | `Ctrl+Alt+Left` |
| Navigate Forward | `Cmd+]` | `Ctrl+Alt+Right` |

<Aside type="tip">
You can customize any keyboard shortcut in **Settings** → **Keymap**. Search for the action name to find and modify its shortcut.
</Aside>

## Customizing Shortcuts

To change a keyboard shortcut:

1. Open **Settings** → **Keymap**
2. Search for the action (e.g., "Rename")
3. Right-click the action
4. Select **Add Keyboard Shortcut**
5. Press your desired key combination
6. Click **OK**

<Aside type="note">
If your new shortcut conflicts with an existing one, the IDE will warn you and let you choose which action to keep.
</Aside>
